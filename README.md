# Electron-Quickstart
**Simple desktop application using the Electron framework**

## Setup
```bash
# Install dependencies
npm install

# Run the app
npm start

# Package it
npm run-script package
```

## Development
See for APIs like Menu: https://github.com/electron/electron-api-demos/releases
